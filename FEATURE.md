# 简介

## 特征

* 让普通人也可以一键编辑的 GPTs
* 只需要学习 markdown 基础语法，即可编排 LLM APPs / GPTs / RAG
* 极其准确且最省 token
* 支持超长文本输出 1000k token + 且逻辑缜密
* 支持各种程序语言进行开发和嵌入
  * js / ts
  * python
  * c++
  * java
  * ...
* 支持各种大模型对接切换/成本控制/统计/速率限制...
  * chatgpt / google gemini / claude ...
  * 文心一言 / 通义千问 / 智谱 / 月之暗面 ...
  * DBRX / Gemma / llama / 百川 ... 
* 支持自然语言编程
  * 类 devin
* 支持通用自动化扩展
  * auto agent 实现自动扩展
  * 可以任意 hook 的标准化通信协议
  * 可直接分析标准网络接口
  * 可克隆任意现实应用和网站
* 编辑的 AI 应用，自动兼容 npm 包管理器
  * 自然语言转为 node 程序，并支持程序化开发
* 自动 AI Commits
* 自动测试
* 自动生成运行过程视频
* 自动发布
  * API
  * 程序
  * 网站 /
